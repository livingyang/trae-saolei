# 扫雷游戏

一款基于Web技术开发的经典扫雷游戏，包含100个精心设计的关卡，难度逐级递增。

## 游戏特性

### 核心玩法
- 左键点击：揭示格子
- 右键点击：标记地雷（显示旗帜）
- 首次点击保护：确保第一个点击的位置不是地雷
- 自动揭示：点击空白格子时，自动揭示周围区域

### 关卡系统
- **100个关卡**，分4个难度阶段
- **初级关卡（1-25关）**：9×9格子，10个地雷
- **中级关卡（26-50关）**：16×16格子，40个地雷
- **高级关卡（51-75关）**：16×30格子，99个地雷
- **终极关卡（76-100关）**：20×30格子，120个地雷

### 游戏功能
- 自动保存游戏进度
- 记录每个关卡的最佳用时
- 关卡解锁系统
- 关卡选择功能
- 实时计时器
- 剩余地雷计数

## 技术栈

- **HTML5**：游戏结构
- **CSS3**：现代化UI设计，响应式布局
- **JavaScript**：游戏逻辑，面向对象编程
- **LocalStorage**：数据持久化存储

## 如何运行

1. 确保已安装Python（用于启动本地服务器）
2. 在项目目录下运行以下命令：

```bash
python -m http.server 8000
```

3. 打开浏览器，访问 `http://localhost:8000`

## 游戏操作

- **开始游戏**：点击"开始游戏"按钮开始当前关卡
- **重置**：点击"重置"按钮重新开始当前关卡
- **选择关卡**：点击"选择关卡"按钮打开关卡选择界面
- **退出**：点击"退出"按钮保存进度并退出游戏

## 胜利条件

揭示所有非地雷格子，且正确标记所有地雷，即可通关当前关卡。

## 失败条件

左键点击到地雷，游戏失败。可以选择重新开始当前关卡或返回关卡选择界面。
